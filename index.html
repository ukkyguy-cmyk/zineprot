序章 / prologue

日常のノイズの中に、置いてきぼりにされた感覚がある。
「言葉にならないまま、残るものがある。」
効率化の網からこぼれ落ちるものに、私たちはもう一度名前を与えなければならない。

第1章 / iwakan

公園のベンチの配置が変わったとき、誰も理由を語らなかった。
「その違和感は、まだ名付けられていない。」
正論で片付けられる前の、胸のつかえを耕すこと。

第2章 / co-en

行政の割り振る『公園』という枠組みを、一度忘れる。
「公園（コウエン）を音（co-en）へ解体する。」
制度の壁を溶かし、ただ人々が共鳴する響きだけをストリートに取り戻す。

第3章 / map

誰のものでもない土地に、勝手に線を引く管理のロジック。
「関わりの地図には、境界線が引けない。」
愛着の深度は、GPSの座標データには決して写らない。

第4章 / pandemic

ウイルスは宣言しない。ただ、静かに広がる。組織の変革も同じだ。「やります」と叫んだ瞬間、免疫が働く。
「日常の隙間に仕込み、気づいたら、もう元には戻れない。」
それが、本当のインストールだ。

第5章 / manual

正しさを記述した紙切れが、現場の息の根を止めていく。
「マニュアルを配った瞬間、思考が停止する。」
余白を恐れる組織が、本当のStreetを殺すのだ。

第6章 / nod

多数決の暴力でもなく、形式的な書類のハンコでもなく。
「合意形成の前に、私たちは頷きの深度を測っていたか。」
言葉の手前にある、身体的な納得のグラデーション。

第7章 / interface

都市を管理するトップダウンのデータ。そこに対抗する個人の記憶。
「都市認知インターフェースは、制度と感情の隙間に立ち上がる。」
見え方によって意味が変わる、あの重なりの設計。

第8章 / layered park

空間はひとつ。しかし体験は一人ひとり異なる。
「物理空間は共有したまま、一人ひとりの認知レイヤーを重ねる。」
既存の空間に、軽やかな未完成のデジタルをそっと漂わせる。

第9章 / mentor

答えを教える指導者は、ストリートには必要ない。
「パブリックメンターは、正解ではなく問いを置いていく。」
人々が勝手に迷い、勝手に耕し始めるための「余白」の守り人。

終章 / epilogue

私たちはどこへ向かうのか。制度に囲い込まれる前に、境界線をあいまいにし続ける。
「余白を管理するな、ただそこに佇め。」
このジンを閉じたとき、あなたの街の景色は、少しだけ書き換わっている。
